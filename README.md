# Rituel

Tracker d'habitudes et carnet personnel. PWA installable, fonctionne hors connexion.

## Stack

HTML + CSS + JavaScript vanilla. Aucune dépendance, aucun build. Persistance en `localStorage`.

## Fonctionnalités

- Composition de rituels (nom, intention, icône, teinte, jours personnalisables)
- Check-in journalier avec séries et constance
- Courbe SVG des 30 derniers jours
- Heatmap calendrier des 13 dernières semaines
- Carnet : notes éditables, cochables, supprimables
- Thème clair / sombre
- Installation sur écran d'accueil (iOS, Android, desktop)
- Fonctionne hors connexion

## Utilisation locale

Ouvrir `index.html` dans un navigateur — c'est tout.

## Déploiement

Servi en statique sur GitHub Pages, Netlify, Vercel, ou n'importe quel hébergeur HTTPS.
